# This project has moved

`Atis.DependencyInjection` is **retired**. It has moved to **Atzonix.DependencyInjection**.

- New repository: https://github.com/atzonix/atzonix-dependency-injection
- New NuGet package: https://www.nuget.org/packages/Atzonix.DependencyInjection

This repository is kept for historical reference only. It is no longer maintained. Its CI/CD
workflows have been disabled (automatic triggers removed; manual `workflow_dispatch` only) to
prevent any accidental publish of the retired package. Please migrate to the Atzonix package.
